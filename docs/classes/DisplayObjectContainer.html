)
DisplayObjectContainer

Defined in src/display/DisplayObjectContainer.js:133

Convenience method for setting the position of the Object.

Parameters:

  • x Number | Array | Vector | Point

    X coord to put the object at. If an Array, Vector, or Point is passed then the y parameter is ignored

  • y Number

    Y coord to put the object at

Returns:

DisplayObjectContainer: Returns itself for chainability

Example:

 obj.setPosition(0) //will set to (0, 0)
     .setPosition(1, 1)
     .setPosition([5, 5])
     .setPosition(new gf.Point(10, 10))
     .setPosition(new gf.Vector(20, 20));
