Boolean

Defined in src\Sprite.js:272

Checks if the name is the active animation

Parameters:

  • name String

    The name of the animation to check if it is currently active

Returns:

Boolean: true if the animation is active, false otherwise.

Example:

 spr.addAnimation('walk-left', new gf.Texture())
     .isActiveAnimation('walk-left'); //false

 spr.setActiveAnimation('walk-left')
     .isActiveAnimation('walk-left'); //true

)
Sprite

Defined in src\Sprite.js:243

Convenience method for setting the position of the Sprite.

Parameters:

  • x Number | Array | Vector | Point

    X coord to put the sprite at. If an Array, Vector, or Point is passed then the y parameter is ignored

  • y Number

    Y coord to put the sprite at

Returns:

Sprite: Returns itself for chainability

Example:

 spr.setPosition(1, 1)
     .setPosition([5, 5])
     .setPosition(new gf.Point(10, 10))
     .setPosition(new gf.Vector(20, 20));
