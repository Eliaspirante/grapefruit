TiledMap Class

Extends Map
Defined in: src\map\tiled\TiledMap.js:1

Tiled map, expects a Tiled TMX file loaded by the gf.loader as the argument. The loader knows to load all textures and other resources when loading a world TMX file, and this expets that to already be done.
