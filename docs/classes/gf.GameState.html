gf.GameState Class

Extends gf.DisplayObjectContainer
Defined in: src/game/GameState.js:1

GameStates are different , controls the entire instance of the game

Constructor

gf.GameState

(
  • [name]
  • [settings]
)

Defined in src/game/GameState.js:1

Parameters:

  • [name] String optional

    The name of this state

  • [settings] Object optional

    All the settings for this game state

    • [gravity] Number optional

      The gravity constant for the physics system (default is 9.87, which is normal Earth gravity)

Example:

 var state = new gf.GameState(game, 'battle');
 state.addChild(battlePlayer);
 state.addChild(enemy);

 game.enableState(state); //or you can use the name from the ctor 'battle'

setPosition

(
  • x
  • y
)
DisplayObjectContainer

Inherited from gf.DisplayObjectContainer: src/display/DisplayObjectContainer.js:134

Convenience method for setting the position of the Object.

Parameters:

  • x Number | Array | Vector | Point

    X coord to put the object at. If an Array, Vector, or Point is passed then the y parameter is ignored

  • y Number

    Y coord to put the object at

Returns:

DisplayObjectContainer: Returns itself for chainability

Example:

 obj.setPosition(0) //will set to (0, 0)
     .setPosition(1, 1)
     .setPosition([5, 5])
     .setPosition(new gf.Point(10, 10))
     .setPosition(new gf.Vector(20, 20));
