AssetLoader Class

Defined in: src\loader\AssetLoader.js:1

The AssetLoader loads and parses different game assets, such as sounds, textures, TMX World JSON file (exported from the Tiled Editor), and Spritesheet JSON files (published from Texture Packer).

Constructor

AssetLoader

(
  • assets
)

Defined in src\loader\AssetLoader.js:1

Parameters:

  • assets Array

    Array of assets to load when .load() is called

Example:

 var loader = new AssetLoader(['/my/texture.png']);
 loader.load();
 //OR
 var loader = new AssetLoader();
 loader.load(['/my/texture.png']);

load

(
  • items
)

Defined in src\loader\AssetLoader.js:88

Starts the loading festivities. If called without any arguments it will load the assets passed in at the ctor. If an array of assets are passed it will load those instead.

Parameters:

  • items Array

    Array of resources to load instead of the object's resources
